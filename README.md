# Online Portfolio

This is a simple online portfolio website showcasing work done for clients.

## Usage

Open `index.html` in your web browser to view the portfolio.

## Customization

- Edit `index.html` to update content.
- Modify `styles.css` for styling changes.
- Add JavaScript functionality in `script.js`.

## Adding Projects

In `index.html`, add more `<div class="project">` elements in the `#projects` section.
